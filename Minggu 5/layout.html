Tutorial Semantik Element untuk Pemula

Semantik element adalah element yang memiliki makna dan tujuan. Tujuannya agar kode HTML mudah dibaca dan tidk ada penyalahgunaan tag. Element semnatik bagus untuk SEO dan juga dapat meningkatkan accessibility.
